SQL

1. Uvod u SQL

SQL je standardni jezik za pohranu, manipuliranje i dohvaćanje podataka u bazama podataka. Ovaj SQL vodič će vas naučiti kako koristiti SQL u raznim sustavima baza podataka. SQL znači Structured Query Language. SQL omogućuje pristup i obradu baza podataka SQL je postao standard Američkog nacionalnog instituta za norme (ANSI) 1986. godine, a Međunarodne organizacije za standardizaciju (ISO) 1987. godine.

Što sve SQL može učiniti?
SQL može izvršavati upite prema bazi podataka, dohvatiti podatke iz baze podataka ,umetnuti zapise u bazu podataka, ažurirati zapise u bazi podataka ,izbrisati zapise iz baze podataka ,stvoriti nove baze podataka ,stvoriti nove tablice u bazi podataka ,stvoriti pohranjene postupke u bazi podataka ,stvoriti poglede u bazi podataka te postaviti dopuštenja na tablice, postupke i prikaze.

Korištenje SQL-a na web mjestu
Za izradu web stranice koja prikazuje podatke iz baze podataka trebat će vam:

  • Program baze podataka RDBMS (tj. MS Access, SQL Server, MySQL)
  • Koristit skriptni jezik na strani poslužitelja, kao što je PHP ili ASP
  • Koristiti SQL za dobivanje željenih podataka
  • Koristiti HTML / CSS za stil stranice
RDBMS
RDBMS je kratica za Relacijski sustav za upravljanje bazama podataka. RDBMS je osnova za SQL i za sve moderne sustave baza podataka kao što su MS SQL Server, IBM DB2, Oracle, MySQL i Microsoft Access. Podaci u RDBMS-u pohranjuju se u objekte baze podataka nazvane tablicama. Tablica je skup povezanih unosa podataka i sastoji se od stupaca i redaka.

2. SQL sintaksa

Tablice baza podataka
Baza podataka najčešće sadrži jednu ili više tablica. Svaka se tablica identificira imenom (npr. "Kupci" ili "Narudžbe"). Tablice sadrže zapise (retke) s podacima. SQL naredbe Većina radnji koje trebate obaviti u bazi podataka obavlja se s SQL naredbama.
Neke od najvažnijih SQL naredbi:

  • CREATE DATABASE - stvara novu bazu podataka
  • CREATE TABLE - stvara novu tablicu
  • INSERT INTO - ubacuje nove podatke u bazu podataka
  • SELECT - izvlači podatke iz baze podataka
  • UPDATE - ažurira podatke u bazi podataka
  • DELETE - briše podatke iz baze podataka
  • ALTER TABLE - mijenja tablicu
  • DROP TABLE - briše tablicu
  • DROP DATABASE - briše indeks
CREATE DATABASE
Izjava CREATE DATABASE koristi se za stvaranje nove SQL baze podataka. 
CREATE DATABASE naziv_baze;
CREATE TABLE
Izraz CREATE TABLE koristi se za stvaranje nove tablice u bazi podataka. 
CREATE TABLE naziv_tablice (
    stupac11 tipPodatka,
    stupac2 tipPodatka,
    stupac3 tipPodatka,
   ....
);
INSERT INTO
Izjava INSERT INTO koristi se za umetanje novih zapisa u tablicu.
INSERT INTO moguće je napisati na dva načina. Prvi način određuje i nazive stupaca i vrijednosti koje treba umetnuti: 
INSERT INTO naziv_tablice (stupac1, stupac2, stupac3)
VALUES (vrijednost1, vrijednost2, vrijednost3);
Ako dodajete vrijednosti za sve stupce tablice, ne morate specificirati nazive stupaca u SQL upitu. Međutim, provjerite je li redoslijed vrijednosti istim redoslijedom kao i stupci u tablici. Sintaksa INSERT INTO bila bi sljedeća: 
INSERT INTO naziv_tablice
VALUES (vrijednost1, vrijednost2, vrijednost3);
SELECT
Izjava SELECT koristi se za odabir podataka iz baze podataka. Vraćeni podaci pohranjuju se u tablicu rezultata, koja se naziva skup rezultata. 
SELECT stupac1, stupac2 
FROM naziv_tablice;
Slijedeći SQL izraz bira sve stupce iz tablice "Kupci": 
SELECT * FROM Kupci;
UPDATE
Izjava UPDATE koristi se za izmjenu postojećih zapisa u tablici. 
UPDATE naziv_tablice
SET stupac1 = vrijednost1, stupac2 = vrijednost2
WHERE condition;
DELETE
Izjava DELETE koristi se za brisanje postojećih zapisa u tablici. 
DELETE FROM naziv_tablice WHERE uvjet;

DELETE FROM Kupci WHERE Ime='Maja';
ALTER TABLE
Izjava ALTER TABLE koristi se za dodavanje, brisanje ili izmjenu stupaca u postojećoj tablici. Izjava ALTER TABLE također se koristi za dodavanje i ispuštanje različitih ograničenja postojećoj tablici. 
ALTER TABLE naziv_tablice
ADD naziv_stupca tipPodatka;

ALTER TABLE Kupci
ADD Email varchar(255);
DROP TABLE
Izjava DROP TABLE koristi se za brisanje postojeće tablice u bazi podataka. 
DROP TABLE naziv_tablice;
DROP DATABASE
Izjava DROP DATABASE koristi se za brisanje postojeće SQL baze podataka. 
DROP DATABASE naziv_baze;

3. SQL Tipovi podataka

Svaki stupac u tablici baze podataka mora imati ime i vrstu podataka. SQL programer mora odlučiti koja će se vrsta podataka pohraniti u svaki stupac prilikom stvaranja tablice. Vrsta podataka je smjernica za SQL za razumijevanje koje se vrste podataka očekuje unutar svakog stupca, a ujedno identificira i način na koji će SQL utjecati na pohranjene podatke. U SQL-u postoje tri glavne vrste podataka: string(niz), numerički te datum i vrijeme.

Numericki tipovi
NUMERIC(l,d) ili DECIMAL(l,d)broj od l znamenki, od cega d decimala. Predznak i decimalna tocka se ne broje u l. d moze biti i 0 ili nenapisan i tada nema decimalne tocke.
SMALLINT, INTEGER mali (16bitni) ili veliki (32bitni) cijeli brojevi.
FLOAT, REAL ili DOUBLE PRECISIONmali ili veliki realni brojevi.
Vremenski tipovi
DATEdatum u formatu yyyy:mm:dd
TIMEvrijeme u formatu hh:mm:ss
DATETIMEdatum i vrijeme u formatu yyyy-mm-dd hh:mm:ss
TIMESTAMPtrenutni datum i vrijeme u formatu yyyymmddhhmmss
Tekstualni tipovi
CHAR(l)niz od tocno l znakova (do 255)
VARCHAR(l)niz od najvise l znakova (do 255)
BLOB ili TEXTniz od najvise 216-1 znakova
LONGBLOB ili LONGTEXTniz od najvise 232-1 znakova
ENUM('s1','s2',...)string iz unaprijed zadane liste 's1','s2',... (najvise 216-1)
SET('s1','s2',...)0, 1 ili vise stringova iz unaprijed zadane liste 's1','s2',... (do 64)

4. SQL join

JOIN se koristi za kombiniranje redaka iz dvije ili više tablica na temelju povezanih stupaca između njih.
Postoje različite vrste JOIN-ova u SQL-u:

  • (INNER) JOIN: Vraća zapise koji u obje tablice imaju odgovarajuće vrijednosti
  • LEFT (OUTER) JOIN: Vraća sve zapise iz lijeve tablice, a podudarne zapise iz desne tablice
  • RIGHT (OUTER) JOIN: Vraća sve zapise iz desne tablice, a podudarne zapise iz lijeve tablice
  • FULL (OUTER) JOIN: Vraća sve zapise ako postoji podudaranje u lijevoj ili desnoj tablici

INNER JOIN Sintaksa 
SELECT naziv_stupca
FROM tablica1
INNER JOIN tablica2
ON tablica1.naziv_stupca = tablica2.naziv_stupca;
WHERE uvjet;
LEFT JOIN Sintaksa 
SELECT naziv_stupca
FROM tablica1
LEFT JOIN tablica2
ON tablica1.naziv_stupca = tablica2.naziv_stupca;
RIGHT JOIN Sintaksa 
SELECT naziv_stupca
FROM tablica1
RIGHT JOIN tablica2
ON tablica1.naziv_stupca = tablica2.naziv_stupca;
FULL JOIN Sintaksa 
SELECT naziv_stupca
FROM tablica1
FULL OUTER JOIN tablica2
ON tablica1.naziv_stupca = tablica2.naziv_stupca;
WHERE uvjet;
WHERE
Klauzula WHERE koristi se za filtriranje zapisa. Upotrebljava se za izdvajanje samo onih zapisa koji ispunjavaju određeni uvjet. 
SELECT * FROM Kupci
WHERE Ime='Maja';
AND, OR, NOT
WHERE može se kombinirati s operaterima AND, OR i NOT. Upotrebljava se za izdvajanje samo onih zapisa koji ispunjavaju određeni uvjet.

Operator AND prikazuje zapis ako su svi uvjeti izdvojeni sa AND i TRUE.
SELECT * FROM Kupci
WHERE Zemlja='Hrvatska' AND Grad='Osijek';
Operater OR prikazuje zapis ako je bilo koji od uvjeta izdvojen sa OR ili TRUE.
SELECT * FROM Kupci
WHERE Grad='Osijek' OR Grad='Zagreb';
Operator NOT prikazuje zapis ako je uvjet NOT TRUE. 
SELECT * FROM Kupci
WHERE NOT Zemlja='Hrvatska';
