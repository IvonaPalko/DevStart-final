HTML

1. Što je to HTML?

HTML predstavlja skraćenicu za "HyperText Markup Language" i on je službeni jezik web stranica. Ono nije računalni programski jezik, HTML se naziva označiteljskim jezikom, koji postavlja upute ili oznake (tagove) oko teksta kako bi internet preglednici znali kako prikazati određeni tekst (ili vizual) na ekranu. Najjednostavnije rečeno, HTML zapravo predstavlja skup instrukcija za internet preglednik. Njegova struktura (hijerarhija) je poprilično jednostavna:

Alati za uređivanje HTML-a

Osnovna "oprema" za rad sa HTML-om je alat za uređivanje koda i teksta; uz naravno, računalo i pristup Internetu.

2. HTML tagovi (elementi)

Kao što je prikazano u izvorima navedenima iznad, HTML je skup oznaka koje definiraju izgled i prikaz sadržaja na web stranici. HTML elementi, odnosno tagovi, su ono od čega je HTML sastavljen. Također, HTML je sastavljen još od elemenata i atributa, o čemu će biti riječi u sljedećem poglavlju. HTML tagovi (elementi) su instrukcije za Internet preglednike o sadržaju pojedinog elementa.
Tagovi (elementi) se sastoje od uglatih zagrada između kojih se nalaze upustva za označivanje teksta. Primjera radi:

<p>Ovo je primjer koda paragrafa čiji bi se sadržaj prikazivao u pregledniku</p>

U ovom primjeru, slovo p okruženo je uglatim zagradama koje predstavljaju oznaku paragrafa. Prva oznaka predstavlja početku oznaku i daje do znanja pregledniku da prikaže sav tekst koji slijedi u obliku paragrafa. Oznaka /p u također uglatnim zagradama predstavlja krajnju oznaku te govori pregledniku da je to kraj paragrafa (kosa crta označava kraj u HTML oznakama).

Iznimke u tagovima

Svi tagovi se pišu sa početnim i završnim tagom (kojeg označava kosa crta /), međutim postoje i iznimke.
Oznaka za liniju te oznaka za prijelog teksta (sljedeći red) su individualne i piše se samo jedna oznaka sa crtom!

<br/> označava prijelom teksta (enter)
<hr/> označava liniju koja se prikaže u redu

3. HTML atributi

HTML elementi mogu imati svoje atribute ili svojstva. Atributi služe kako bi pružili dodatne informacije o pojedinom elementu. Oni obično imaju naziv i vrijednost, a u samom kodu se ispisuju naziv=“vrijednost”
Atributi su načelno opcionalni, odnosno možemo ih koristiti međutim nije greška ukoliko ih izostavimo.
Na primjer, naslov pojedine stavke (npr. slike na web stranici) će se prikazati ukoliko prinesemo miš na nju. Taj naslov je dodani atribut, koji nije morao postojati, a dodao se unutar elementa img kao naslov (title=“Naslov slike”).

Nešto više o atributima

Pojedini elementi nisu upotrebljivi bez atributa koji ih pobliže određuje, kao naprimjer element a. On označava link (hipervezu) te se nije funkcionalan ukoliko mu se ne dodijeli atribut, što je u ovom slučaju href gdje se upisuje putanja na koju vodi.
Također, element img mora imati izvor (atribut src) odakle povlači sliku.

4. HTML DOM (Document Object Model)

Kada se web stranica učita, Internet preglednik kreira DOM (Document Object Model). HTML DOM definira standard za pristup i manipulaciju HTML dokumentima. DOM definira objekte i svojstva svih dokumentiranih elemenata, kao i metode kojima im se pritupa. 5. Liste

Za urednost teksta, kao i pisanog na papiru te u Wordu, važno je imati organizirane i strukturirane liste i na web stranicama. Liste su uglavnom kratke, numerirane ili imaju točke za nabrajanje te osiguravaju da dijelovi teksta budu brzo i lako uočljivi od stranice čitaoca. HTML nudi više vrsta listi, ali se dvije izdvajaju kao najbitnije: numerirana lista te lista za nabrajanje. Numerirane liste se još nazivaju i uređene liste, dok s liste sa znakovima za nabrajanje (nenumerirane) nazivaju i neuređenim listama.

6. Hiperveze

Hiperveza (URL) predstavlja skup instrukcija koje treba napraviti kako bi se izvršili zahtjevi određenog izvora. Sadrže određene adrese pomoću kojih se pronalaze dokumenti, videi, audio datoteke, tekstovi te objekti iz svih krajeva svijeta. Hiperveze dozvoljavaju da skačete a jedne na drugu stavku i da vidite izvore na toj adresu, pri čemu nije važno gdje se nalazite. Opći naziv za ove adrese je URL, odnosno engl. Uniform Resource Locators. Svaka web stranica na svijetu, svaki izvor na mreži, bilo da se radi o mješavini teksta i multimedije ili grafike, ima svoju jedinstvenu URL adresu, koja predstavlja strogo definiranu adresu pojedinog web izvora.
Element za hipervezu je <a> </a> koji sadrži obavezni atribut href.

Upisivanje u alatu za uređivanje koda bi bilo sljedeće: <a href=“www.DevStart.com”>DevStart</a>

Dodatno:

Upoznati smo sa činjenicom da nije dobro ukoliko korisnik dođe na naš web te klikne na link koji ga odmah vodi na neku drugu web stranicu, dok mi ostajemo bez korisnika. Kako bi se to izbjeglo, moguće je dodati taget=“_blank” atribut i vrijednost i time usmjeriti web stranicu u novootvorenu karticu, bez zatvaranja našeg weba.

7. Slike

Kako bismo dali uputu pregledniku da prikaže sliku, potrebno je osigurati dvije stvari. Prvo, potrebna je slika koju želimo prikazati, što se radi pomoću atributa src oznake <img>. Da bi se slika prikazala, vrijednost mora biti ispravna URL adresa, apsolutna ili relativna. Druga potrebna stvar jeste mjesto na web stranici gdje će slika biti smještena, što se također radi pomoću postavljanja elementa <img>.

Oznaka <img> nije element nego vrijedi kao samostalna oznaka, što znači da nema krajnje crte, niti oznake koja zatvara za blok. Umjesto toga se koristi kosa crta na samom kraju te iste oznake:
<img src=“URL ili putanja slike” alt=“Naziv slike”/>

8. Tablice

HTML tablice izrađuju se pomoću oznaka koje formiraju redove i ćelije. Specifične ćelije mogu se prostirati i u redovima i u stupcima te tako kreirati veće dijelove prostora.
Osnovni element tablice je <table> koji daje uputu pregledniku da slijede elementi za redove i stupce. Nadalje dolaze redovi sa elemetom <tr>. Svaki red HTML tablice sadrži niz horizontalnih ćelija sa podacima u kojima se nalaze podaci. Ćelije se označavaju sa elementom <td>. Pored hijerarhije tablice koja je navedena, a tiče se samo redova i ćelija, postoji još jedna struktura tablice (thead, tbody, tfoot):

Hijerarhija tablice

Hijerarhija elemenata je označena elementom <table>...</table>, koji sadrži sve ostale elemente tablice. Element za redove tablice (<tr>...<tr>) dolazi unutar table elementa, a unutar njega upisujemo ćelije s podacima (<td>...</td>).
