HTML

Što je to HTML?

HTML predstavlja skraćenicu za "HyperText Markup Language" i on je službeni jezik web stranica. Ono nije računalni programski jezik, HTML se naziva označiteljskim jezikom, koji postavlja upute ili oznake (tagove) oko teksta kako bi internet preglednici znali kako prikazati određeni tekst (ili vizual) na ekranu. Najjednostavnije rečeno, HTML zapravo predstavlja skup instrukcija za internet preglednik.

Poveznice na uvod u HTML

Alati za uređivanje HTML-a

Osnovna "oprema" za rad sa HTML-om je alat za uređivanje koda i teksta; uz naravno, računalo i pristup Internetu. P.S. većina operativnih sustava sadrže alate za uređivanje teksta i koda, a oni mogu, za početak, biti sasvim dovoljni za jednostavne web stranice.
