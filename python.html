1. Uvod u Python programski jezik

Python je popularan programski jezik. Stvorio ga je Guido van Rossum, a objavio 1991. godine. Koristi se za: web razvoj (na strani poslužitelja), razvoj softvera, matematiku, sistemsko skriptiranje. Python je dizajniran kako bi bio razumljiv i ima neke sličnosti s engleskim jezikom uz utjecaj matematike. Python koristi nove retke za dovršavanje naredbe, za razliku od ostalih programskih jezika koji često koriste zarez ili zagrade. Python se za definiranje opsega oslanja na uvlačenje pomoću razmaknica; poput opsega petlje, funkcija i klasa. Ostali programski jezici često u tu svrhu koriste vitičaste zagrade.

Zašto koristiti Pyhton?

  • Python radi na različitim platformama (Windows, Mac, Linux, Raspberry Pi, itd.)
  • Ima jednostavnu sintaksu sličnu engleskom jeziku
  • Python ima sintaksu koja omogućava programerima da pišu programe s manje redaka nego neki drugi programski jezici
  • Python se pokreće na sustavu za tumačenje, što znači da se kôd može izvršiti čim je napisan. To znači da prototipiranje može biti vrlo brzo
  • Python se može tretirati na proceduralni način, objektno orjentiran ili funkcionalan način.
Instalacija Pythona

Koristimo Python verziju 3.0.

Na nekim računalima Python je već instaliran. Da biste provjerili imate li instaliran Pyhton na Windows računalu, potražite Pyhton na početnoj traci ili upišite sljedeće u naredbeni redak (cmd.exe): 
C:\Users\Vaše ime>python --version
U slučaju da je potrebno instalirati Python na računalo, pratite korake na linku.

2. Python Hello World

Python je interpretirani programski jezik, to znači da kao programer pišete datoteke Python (.py) u uređivačima teksta i zatim te datoteke stavljate u tumač pythona koji će se izvršiti. Način pokretanja python datoteke u naredbenom retku: 

C:\Users\Your Name>python helloworld.py
Gdje je "helloworld.py" ime vaše python datoteke. Kreirajte vašu prvu python datoteku, zvanu helloworld.py, koja se može napisati u bilo kojem uređivaču teksta. 
 print("Hello, World!")
Nakon toga, spremite datoteku, otvorite naredbeni redak, pomaknite se do direktorija u koji ste spremili datoteku i pokrenite: 
C:\Users\Your Name>python helloworld.py
izlaz bi trebao glasiti Hello World .
Za testiranje kratke količine koda u pythonu ponekad je najbrže i najlakše ne napisati kôd u datoteku. To je moguće jer Python može biti pokrenut kao naredbeni redak. Ili, ako python naredba nije uspjela, možete pokušati s "py". Kako biste napustili sučelje naredbenog retka python, jednostavno unesite exit() u naredbeni redak.

3. Pyhton Sintaksa

Dijelovi Python koda
U drugim programskim jezicima uvlačenje u kodu služi kako bi kod bio čitljiv, dok je uvlačenje u Pythonu vrlo važno.
Python koristi uvlačenje za označavanje bloka koda. Ne koristi znak točka-zarez (;). 

 if 5 > 2:
       print("Pet je veće od dva!")
Python će vam javiti grešku ukoliko preskočite uvlačenje (gore označeno). Broj uvlačenja (razmaka) ovisi o programeru, ali mora biti barem jedan. Morate koristiti isti broj razmaka u istom bloku koda, u protivnom će vam Python javiti grešku.

4. Tipovi podataka

U Pythonu postoje dvije numeričke vrste:

  • int
  • float
Varijable numeričkih vrsta nastaju kada im dodijelite vrijednost. Za provjeru tipa bilo kojeg objekta u Pythonu, upotrijebite funkciju type (): 
print(type(x))
Int, ili cijeli broj, je cijeli broj, pozitivan ili negativan, bez decimala, neograničene duljine.
Float, ili "broj s pomičnim zarezom" je broj, pozitivan ili negativan, koji sadrži jedan ili više decimala.
Float može biti i znanstveni brojevi sa slovom "e" koji označavaju snagu 10.

Možete mijenjati tipove podataka pomoću metoda int () i float (): 
x = 1 # int
y = 2.8 # float


#promjena iz int u float:
a = float(x)

#promjena iz float u int:
b = int(y)



print(a)
print(b)


print(type(a))
print(type(b))
Python nema funkciju random() za izradu slučajnih brojeva, ali Python ima ugrađeni modul koji se zove random i koji se može koristiti za izradu slučajnih brojeva. 
import random

print(random.randrange(1,10))
Ponekad možete tip podatka specificirati na varijablu. To se može učiniti castingom . Python je objektno orijentirani jezik i kao takav koristi klase za definiranje tipova podataka, uključujući i njegove primitivne tipove.

Casting u pythonu se stoga vrši pomoću konstruktorskih funkcija:
  • int () - konstruira cijeli broj , float (zaokruživanjem prema donjem na cijeli cijeli broj) ili niza string (pod uvjetom da niz predstavlja cijeli broj)
  • float () - konstruira float broj iz cjelobrojnog doslovnog, plutajućeg doslovnog ili niza literalnog (pod uvjetom da niz predstavlja float ili cijeli broj)
  • str () - konstruira niz iz širokog spektra podataka, uključujući nizove, cijele brojeve i float brojeve

String u pythonu okruženi je jednostrukim ili dvostrukim navodnicima. 
'jabuka' je isto što i "jabuka".

Možete prikazati string pomoću funkcije print (): 
print("Hello")
print('Hello')
Dodjeljivanje niza varijabli vrši se s nazivom varijable, nakon čega slijedi znak jednakosti i niza: 
a = "Hello"
print(a)
Nizu u više linija možete dodijeliti varijable pomoću tri navodnika : 
a = """Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua."""
print(a)
mogu biti i jednostruki: 
a = '''Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua.'''
print(a)
Kao u mnogim drugim popularnim programskim jezicima, i nizovi Pythona su nizovi bajtova koji predstavljaju znakove unicode. Međutim, Python nema znakovni tip podataka, jedan znak je jednostavno niz duljine 1. Uglate zagrade mogu se koristiti za pristup elementima niza. 
a = "Hello, World!"
print(a[1])
Možete vratiti niz znakova koristeći sintaksu rezanja. Navedite početni indeks i završni indeks, odvojene dvotočkom, kako biste vratili dio niza. 
a = "Hello, World!"
print(b[2:5])
Upotrijebite negativne indekse da biste započeli slice s kraja niza: 
a = "Hello, World!"
print(b[-5:-2])
Da biste dobili duljinu niza, upotrijebite funkciju len (). 
a = "Hello, World!"
print(len(a))
Python ima skup ugrađenih metoda koje možete koristiti na nizu.
  • Metoda strip () uklanja bilo koji prazni prostor s početka ili kraja
  • Metoda lower () vraća niz u malim slovima
  • Metoda upper () vraća niz u velikim slovima
  • Metoda replace () zamjenjuje niz drugim nizom
  • Metoda split () dijeli niz na podnaslove ako pronađe separatore

Boolean
Metoda bool () koristi se za vraćanje ili pretvaranje vrijednosti u logičku vrijednost, tj. TRUE ili FALSE, koristeći standardni postupak ispitivanja. 
bool([x])
# Vraća False jer je x False 
x = False
print(bool(x)) 
  
# Vraća True jer je x True 
x = True
print(bool(x))

5. Python Varijable

Varijable su spremnici za pohranjivanje podataka, odnosno prostor u memoriji koji je povezan ljudima smislenim imenima. Za razliku od ostalih programskih jezika, Python nema naredbu za proglašavanje varijable. Varijabla se kreira u trenutku kada joj prvi put dodijelite vrijednost. 

x = 5
y = "Banana"
print(x)
print(y)
Varijable se ne moraju deklarirati određenim tipom, a mogu čak i promijeniti tip nakon što su postavljene. 
x = 4 # x je sada tip int
x = "Jabuka" # x je sad tip str
print(x)

Varijable String mogu se deklarirati ili korištenjem jednostrukih ili dvostrukih navodnika:
x = "Mačka"

jednako je

x = 'Mačka'

Imenovanje varijabli

Varijabla može imati kraće ime (poput x i y) ili opisnije ime (starost, naziv automobila, ukupni_broj).
Pravila za Python varijable:
  • Naziv varijable mora započeti slovom ili znakom podvlake
  • Ime varijable ne može započeti brojem
  • Naziv varijable može sadržavati samo alfa-numeričke znakove i podvlake (A-z, 0-9 i _)
  • Nazivi varijabli razlikuju velika i mala slova (DOB, dob i Dob su tri različite varijable)

Dodijeljivanje vrijednosti varijabli

Python vam omogućava dodijeliti vrijednosti više varijabli u jednom retku: 
x, y, z = "Jabuka", "Banana", "Jagoda"
print(x)
print(y)
print(z)
A istu vrijednost možete dodijeliti više varijabli u jednom retku: 
x, y, z = "Jabuka"
print(x)
print(y)
print(z)
Naredba za ispis Pythona koristi se za ispis varijabli. Za kombiniranje teksta i varijable Python koristi znak +: 
x = "odličan"
print("Python je " + x)
Možete upotrijebiti i znak + da biste dodali jednu varijablu u drugu varijablu: 
z = "Pyhton je"
x = "odličan"
y = z + x
print(y)
Za brojeve, znak + djeluje kao matematički operator: 
z = 7
x = 2
print(x + z)
Ako pokušate kombinirati listu i broj, Python će vam javiti grešku.

Globalne varijable
Varijable koje su stvorene izvan funkcije (kao u svim gore navedenim primjerima) poznate su kao globalne varijable. Globalne varijable mogu koristiti svi, kako unutar funkcija tako i izvan njih. Ako unutar funkcije stvorite varijablu s istim imenom, ova će varijabla biti lokalna i može se koristiti samo unutar funkcije. Globalna varijabla s istim nazivom ostat će onakva kakva je bila, globalna i s izvornom vrijednošću.

Globalna ključna riječ
Kada stvorite varijablu unutar funkcije, obično je lokalna i može se koristiti samo unutar te funkcije. Za stvaranje globalne varijable unutar funkcije možete koristiti globalnu ključnu riječ global . 
def mojafunk():
 global x
 x = "top"

 mojafunk()

print("Python je " + x)
Također, koristite ključnu riječ globalno ako želite promijeniti globalnu varijablu unutar funkcije.

6.Liste

Liste se koriste za pohranjivanje više vrijednosti u jednu varijablu 

gradovi = ["Zagreb", "Rijeka", "Osijek"]
Lista je posebna varijabla koja može sadržavati više vrijednosti odjednom. Ako imate popis predmeta (na primjer popis imena gradova), pohranjivanje gradova u jednu varijablu može izgledati ovako: 
grad1 = "Zagreb"
grad2 = "Rijeka"
grad3 = "Osijek"
Međutim, što ako želite proći kroz gradove i pronaći određeni? A što ako niste imali 3 grada, već 300? Rješenje je lista! Lista može sadržavati mnoge vrijednosti pod jednim nazivom, a vrijednostima možete pristupiti pozivanjem na indeksni broj. 
x = gradovi[0]
Izmijenite vrijednost prve stavke liste: 
gradovi[0] = "Split"
Pomoću metode len () vraćate duljinu liste (broj elemenata u listi). 
x = len(gradovi)
Možete koristiti for in petlju da biste prešli kroz sve elemente liste. 
 for x in gradovi:
  print(x)
Možete koristiti metodu append () da biste dodali element u listu. 
  gradovi.append("Opatija")
Možete koristiti metodu pop () ili remove () da biste obrisali element u listi. 
  gradovi.pop(1)
  gradovi.remove("Rijeka")

Python ima skup ugrađenih metoda koje možete koristiti na listama.

  • append () Dodaje element na kraj popisa
  • clear () Uklanja sve elemente s popisa
  • copy () Vraća kopiju popisa
  • count () Vraća broj elemenata s navedenom vrijednošću
  • exte () Dodajte elemente popisa (ili bilo kojeg ponovljivog) na kraj trenutnog popisa
  • index () Vraća indeks prvog elementa s navedenom vrijednošću
  • insert () Dodaje element na navedenoj poziciji
  • pop () Uklanja element na navedenom položaju
  • remove () Uklanja prvu stavku s navedenom vrijednošću
  • reverse () Obrne redoslijed popisa
  • sort () Poredava popis
