JAVA

1. Uvod u JAVA programski jezik

Java je programski jezik.Java se koristi za razvoj mobilnih aplikacija, web aplikacija, desktop aplikacija i još mnogo toga. Stvorena je 1995. godine. U vlasništvu je Oraclea, a više od 3 milijarde uređaja koristi Javu. Zašto koristiti Java programski jezik?

  • Java radi na različitim platformama (Windows, Mac, Linux, Raspberry Pi, itd.)
  • To je jedan od najpopularnijih programskih jezika na svijetu
  • Jednostavan je za učenje i jednostavan za korištenje
  • Otvoren je i besplatan
  • Siguran je, brz i moćan
  • Ima veliku podršku zajednice (desetine milijuna programera)
Instalacija Jave
Na nekim računalima Java je već instalirana. Da biste provjerili imate li instaliranu Javu na Windows računalu, potražite Javu na početnoj traci ili upišite sljedeće u naredbeni redak (cmd.exe):
C:\Users\Your Name>java -version
Ako je Java instalirana, vidjet ćete ovako nešto (ovisno o verziji):
java version "11.0.1" 2018-10-16 LTS Java(TM) SE Runtime Environment 18.9 (build 11.0.1+13-LTS) Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.1+13-LTS, mixed mode)
U slučaju da je potrebno instalirati Javu na računalo, pratite korake na linku.

2. JAVA Hello World

U Javi svaka aplikacija započinje s nazivom klase i ta klasa mora odgovarati nazivu datoteke. Napravimo našu prvu Java datoteku, nazvanu MojaKlasa.java, koja se može kreirati u bilo kojem uređivaču teksta (poput Bilješki/Notepad). Datoteka bi trebala sadržavati poruku "Hello World" koja je napisana na sljedeći način:

public class MojaKlasa {
public static void main(String[] args) {
System.out.println("Hello World");
}
}
Ne brinite ako ne razumijete gornji kôd - o tome ćemo detaljno govoriti u kasnijim poglavljima. Za sada se usredotočite na to kako pokrenuti kod iznad. Spremite kod u bilješkama kao "MojaKlasa.java". Otvorite naredbeni redak (cmd.exe), pomaknite se do direktorija u kojem ste spremili datoteku i upišite "javac MojaKlasa.java":
javac MojaKlasa.java

To će sastaviti(kompajlirati) vaš kôd. Ako u kodu nema pogrešaka, naredbeni redak prevest će vas do sljedećeg retka. Sada upišite "java MojaKlasa" za pokretanje datoteke:
java MojaKlasa
Izlaz treba glasiti:
Hello World
Čestitamo, napisali ste i izveli svoj prvi Java program.

3. JAVA Sintaksa

Svaka linija koda koja se pokreće u Javi mora biti unutar klase. U našem primjeru imenovali smo klasu MyClass. Razred uvijek treba započeti velikim slovom. Napomena: Java razlikuje velika i mala slova: "MyClass" i "myclass" imaju različito značenje. Naziv java datoteke mora odgovarati imenu klase. Prilikom spremanja datoteke spremite je koristeći naziv klase i dodajte ".java" na kraj imena datoteke.

Glavna metoda / main metoda

Glavna metoda [main () method] je potrebna i vidjet ćete je u svakom Java programu:
public static void main(String[] args)
Bilo koji kod unutar glavne metode će se izvršiti. Kao početnik, ne morate razumjeti ključne riječi prije i poslije glavne metode. Za sada samo zapamtite da svaki Java program ima naziv klase koji mora odgovarati nazivu datoteke i da svaki program mora sadržavati glavnu metodu [main () method].

System.out.println()

Unutar glavne metode možemo koristiti metodu println () za ispis retka teksta na zaslon:
public static void main(String[] args) {
System.out.println("Hello World");
}

4. JAVA Varijable

Varijable su spremnici za pohranjivanje podataka, odnosno prostor u memoriji koji je povezan ljudima smislenim imenima. U Javi postoje različite vrste varijabli, na primjer:

  • String - pohranjuje tekst, poput "Hello". Vrijednosti niza okružene su dvostrukim navodnicima
  • int - pohranjuje cijele brojeve, bez decimalnih brojeva, poput 123 ili -123
  • float - pohranjuje brojeve s decimalama, kao što su 19,99 ili -19,99
  • char - pohranjuje pojedinačne znakove, poput 'a' ili 'B'. Vrijednosti Char okružene su jednostrukim navodnicima
  • boolean - pohranjuje vrijednosti s dva stanja: true ili false

Deklariranje (stvaranje) varijabli

Da biste stvorili varijablu, morate navesti vrstu i dodijeliti joj vrijednost:
vrsta varijable = vrijednost;
gdje je vrsta jedna od Java vrsti (poput int ili String), a varijable je naziv varijable (kao što su x ili ime). Znak jednakosti koristi se za dodjeljivanje vrijednosti varijabli.

Prikaz varijabli

Metoda println () često se koristi za prikaz varijabli. Za kombiniranje teksta i varijable upotrijebite znak + :
String name = "Barbara";
System.out.println("Hello " + name);
Možete upotrijebiti i znak + da biste dodali jednu varijablu u drugu varijablu:
String ime = "Barbara ";
String prezime = "Barbić";
String punoIme = ime + prezime;
System.out.println(fullName);
Za numeričke vrijednosti, znak + djeluje kao matematički operator (primijetite da ovdje koristimo int (integer) varijable):
int x = 2;
int y = 9;
System.out.println(x + y); // Prikazuje se vrijednost x + y
Deklariranje više varijabli
Da biste deklarirali više od jedne varijable iste vrste, koristite popis odijeljen zarezom:
int x = 2, y = 9, z = 20;
System.out.println(x + y + z);
Java identifikatori
Sve Java varijable moraju biti identificirane s jedinstvenim imenima. Ta jedinstvena imena nazivaju se identifikatori. Identifikatori mogu biti kratka imena (poput x i y) ili više opisna imena (dob, zbroj, ukupna vrijednost). Opća pravila za izgradnju imena varijabli (jedinstvenih identifikatora) su:
  • Imena mogu sadržavati slova, znamenke, podvlake i znak dolara
  • Imena bi trebala započeti slovom
  • Imena također mogu započeti s $ i _
  • Imena razlikuju velika i mala slova ("mojaVarijabla" i "mojavarijabla" su različite varijable)
  • Imena bi trebala započeti malim slovom i ne mogu sadržavati razmak
  • Rezervirane riječi (poput Java ključnih riječi, poput int ili String) ne mogu se upotrebljavati kao imena

5. Tipovi podataka

Kao što je objašnjeno u prethodnom poglavlju, varijabla u javi mora biti određene vrste podataka:

  • int mojBroj = 5;
  • float mojFloatBroj = 5.99f;
  • char mojeSlovo = 'E';
  • boolean mojBool = true;
  • String myText = "Hello World";

Vrste podataka dijele se u dvije skupine:
  • Primitivni tipovi podataka - uključuju byte, short, int, long, float, double, boolean i char
  • Neprimitivni tipovi podataka - poput stringa, niza i klasa

Primitivni tip podataka


Primitivni tip podataka određuje veličinu i vrstu varijabilnih vrijednosti i nema dodatnih metoda.
U Javi postoji osam primitivnih vrsta podataka:
  1. short: cijeli broj (-32,768 do 32,767 )
  2. int: cijeli broj (-231 do 231-1)
  3. long: cijeli broj (-263 do 263-1)
  4. float: decimalni broj
  5. double: decimalni broj
  6. boolean: logička vrijednost (true/false)
  7. char: znak (ukupno mogućih 65,535 znakova)

Neprimitivne vrste podataka


Neprimitivni tipovi podataka nazivaju se referentni tipovi jer se odnose na objekte. Glavna razlika između primitivnih i neprimitivnih vrsta podataka je:
  • Primitivne vrste su unaprijed definirane (već definirane) u Javi.
  • Neprimitivne vrste kreira programer, a ne definira ih Java (osim String).
  • Neprimitivni tipovi mogu se koristiti za pozivanje metoda za izvođenje određenih operacija, dok primitivni tipovi ne mogu.
  • Primitivni tip uvijek ima vrijednost, dok neprimitivni tipovi mogu biti bez vrijednosti.
  • Primitivni tip započinje malim slovom, dok neprimitivni tipovi počinju velikim slovom.
  • Veličina primitivnog tipa ovisi o vrsti podataka, dok su neprimitivne vrste sve iste veličine.

6. Nizovi

Nizovi se koriste za pohranjivanje više vrijednosti u jednu varijablu, umjesto deklariranja zasebnih varijabli za svaku vrijednost.
Da biste deklarirali niz, definirajte vrstu varijable s uglatim zagradama:

String[] gradovi;
String[] gradovi = {"Zagreb", "Osijek", "Rijeka", "Split"};
int[] mojiBrojevi = {1, 2, 3};
Niz gradovi ima 4 elementa, dok niz mojiBrojevi ima tri elementa. Elementu niza pristupate pozivanjem na indeksni broj. Broj indeksa u niz kreće od nule. Ova izjava pristupa vrijednosti prvog elementa u nizu gradovi:

String[] gradovi = {"Zagreb", "Osijek", "Rijeka", "Split"};
System.out.println(gradovi[0]);

Da biste promijenili vrijednost određenog elementa, koristite indeksni broj:

String[] gradovi = {"Zagreb", "Osijek", "Rijeka", "Split"};
gradovi[0] = "Opatija";
System.out.println(gradovi[0]);
Izlaz prikazuje Opatija umjesto Zagreb.
Da biste saznali koliko elemenata ima niz, upotrijebite svojstvo dužine:

String[] gradovi = {"Zagreb", "Osijek", "Rijeka", "Split"};
System.out.println(gradovi.length);
Izlaz prikazuje vrijednost 4 jer se u nizu nalaze 4 elementa.

Petlje

Petlja je ponavljanje do izvršenja uvjeta. U javi razlikujemo :
  • for
  • while
  • do while
  • for each
For
for (int i = 0; i < 0 i++);
Prikazana petlja kreće od broja 0 (jer se prvi element nalazi na indeksu 0) i petlja se vrti izvodi se sve dok je i manji od broja 10. Petlja će se prekinuti kada i bude imao vrijednost 10. ++ je operator koji definira uvjet promjene. On se naziva inkrement jer uvećava vrijednost za jedan.
while
int i = 0;
while (i < 5) {
System.out.println(i);
i++;
}
While je petlja koja se izvršava sve dok se ne zadovolji uvjet. While petlja uvjet provjerava na početku te je moguće da se nikada niti ne uđe u while petlju. Petlja iznad vrti se sve do i ne dobije vrijednost 5.
do while
int i = 0;
do {
System.out.println(i);
i++;
}
while (i < 5);
Do while isti je kao i while petlja. Jedina razlika jest da se do while petlja izvodi barem jednom jer se uvjet provjerava na kraju petlje.
for each
String[] gradovi = {"Zagreb", "Osijek", "Rijeka", "Split"};
for (String i : gradovi) {
System.out.println(i);
}
For each specifičan je oblik for petlje. Korištenje for each petlje omogućuje prolaz kroz sve elemnte niza. Ona se može prekinuti i nastaviti (break i continue).
