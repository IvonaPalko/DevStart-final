JavaScript

1. Uvod u JavaScript programski jezik

JavaScript predstavlja najpopularniji skriptni jezik kojim možemo kreirati interaktivne web stranice. Interaktivne web stranice omogućavaju međudjelovanje s korisnikom, upravljanje web preglednikom ili dinamično kreiranje web stranice. JavaScript se izvodi na klijentskom računalu i može se izvoditi u svim novijim verzijama web preglednika.
Standard JavaSrcipta je ECMA-262.

Neke od prednosti JavaScripta su:

  • Manja potreba za komunikaciju sa serverom. Može se provjeriti ispravnost podatka prije slanja stranice na server. To smanjuje promet prema serveru.
  • Trenutni odziv korisniku. Korisnik ne treba čekati da se stranica ponovno učita kako bi vidio je li zaboravio unijeti neki podatak.
  • Povećana interaktivnost. Moguće je kreirati sučelje koje reagira na korisnikove akcije tipkovnice ili miša.
  • Bogatije sučelje. Možete koristiti JavaScript za uhvati i pusti komponente ili animirati bogatije sučelje prema korisniku.
JavaScript, sam po sebi, je komplementaran jezik, što znači da nije nemoguće napisati cijelu aplikaciju samo u JavaScripti. JavaScript predstavljen je slovom J u akronimu AJAX (Asynchronous JavaScript and XML), međutim, pored toga, JS je svakodnevni jezik koji omogućava očekivanu interaktivnost web stranice. JavaScript može izvršavati brojne zadatke na klijentskoj strani aplikacije (padajući izbornici, pomoć pri ispunjavanju obrazaca…).

Primjena JavaScripta obuhvaća široki spektar, iako se ono odvija samo u okviru web preglednika. JS se može koristiti za potrebe:

  • vizualnog bogaćenja HTML dokumenta (animacije, interaktivnost, stilizacije)
  • pristupa sadržaju (mogu se odabrati bilo koji HTML elementi, atributi ili njihov sadržaj)
  • mijenjanja sadržaja - utjecaj na HTML strukturu bez ponovnog učitavanja dokumenta
  • kreiranja različitih web aplikacija - igrice, glazbeni programi i sl.
  • provjere obrazaca - validacija kontrola obrazaca

2. JavaScript sintaksa

Kao i svaki drugi jezik, i JavaScript ima svoj rječnik i sintaksu, odnosno pravila za pisanje. Njegova sintaksa je bazirana na programskom jeziku C.


Pisanje identifikatora

Identifikatori su nazivi kojima se imenuju varijable, funkcije, objekti i nazivi svojstava objekata. Pravila za formiranje ispravnih naziva za identitifikatore u JavaScriptu su ista kao u Javi. Prvi znak mora biti: slovo donja crta ili znak dolar ($). Dakle, ne smije biti brojka iii neki drugi znak. Svaki sljedeći znak može biti slovo, donja crta, znak dolar, ili brojka.


Izjave (engl. statements)

U JavaScriptu nije obavezno završavanje izjave točkom i zarezom već je opcionalno, i njeno nepostojanje uglavnom neće izazvati grešku pri izvršavanju programa, međutim programerska praksa pitanja točke i zareza je toliko jaka da je ostala i u JS-u.
Određene strukture jezika ne zahtjevaju upotrebu točke i zareza, dok je ona obavezna ako se više izjava ispisuju odjednom. Primjerice: 

      var pozdrav = "Dobar dan!";
      document.write(pozdrav);


Ključne riječi

Postoje određene riječi u JavaScriptu koje se vode kao ključne riječi jer imaju određenu ulogu (for, if, var, case… ). Ove riječi se ne smiju koristiti kao identifikatori jer već same po sebi imaju namjenu.


Blokovi koda

Određene strukture u JavaScriptu sadrže blokove koda, kao to su petlje i funkcije. Blokovi koda odvajaju se vitičastim zagradama, odnosno kod koji se nalazi među njima je blok. Npr.: 

      if (a == b) {document.write('Jednako je.');}
U primjeru iznad cijeli kod između vitičastih zagrada predstavlja blok koda. Kodni blok može sadržati proizvoljnu količinu koda u sebi. Važno je obratiti pažnju da kada se jednom otvori vitičasta zagrada na početku bloka, na kraju bloka se ona mora i zatvoriti.


Velika i mala slova

U JavaScript jeziku nije svejedno jesu li slova mala ili velika već on prepoznaje razliku. Usvojeno je pravilo da nazivi identifikatora za varijable, funkcije, nizove itd. počinju malim slovom, a da nazivi konstruktora počinju velikim slovom. Ukoliko se identifikator sastoji od više od jedne riječi, koristi se tzv. camelCase, odnosno spajanje te dvije riječi uz veliko početko slovo druge riječi.


Komentari

Komentari se mogu pisati u jednom redu ili u više redova. Također, komentiranje može poslužiti i pri ispitivanju i traženju grešaka. Pisanje komentara u jednom redu vrši se pomoću znakova //, dok se komentiranje više redova vrši pomoću skupa znakova /* */, isto kao u CSS-u.

3. JavaScript varijable

Varijable služe da se uskladište podaci u memoriju i da se koriste u programu. Naime, nije dovoljno varijablu samo deklarirati, već joj dodati i vrijednosti. Definiranje varijable se vrši naredbom za deklariranje, a ona je var - sintaksa je sljedeća: 

 var nazivVarijable;
Naziv same varijable potrebno je osmisliti tako da bude smisleno u odnosu na njenu ulogu ili sadržaj. Deklariranje varijable kreira praznu varijablu, dok dodavanje vrijednosti čini inicijalizaciju varijable: 
  var varijabla;                    // deklariranje varijable
  varijabla = "neki tekst"          // inicijalizacija varijable
Nakon deklariranja varijable više nije potrebno koristiti riječ var.

Pravila za JavaScript varijable:
  • prvi karakter mora biti slovo, donja crta ili znake $
  • ostali karakteri mogu biti slova, brojke, donja crta ili $

4. JavaScript tipovi podataka

Varijable mogu pohraniti podatke različitih vrsta, a različite vrste podataka mogu činiti različite stvari.
PHP podržava sljedeće vrste podataka:

  • String
  • Boolean
  • Number
  • Object
  • Array
  • NULL i undefined

String

String je niz znakova, poput "Hello world!". String može biti bilo koji tekst unutar citata. Možete koristiti jednostruke ili dvostruke navodnike. 
var primjer = "tekst";
var primjerDrugi = 'isto tekst';

Boolean

Boolean predstavlja dva moguća stanja: TRUE ili FALSE. 
var primjer3 = true;
var primjer4 = false;

Number (broj)

Brojevi se koriste kao brojevni literali (doslovne vrijednosti). Nad brojevima koje je moguće koristiti moguće je vršiti matematičke operacije upotrebom aritmetičkih operatora (+, -, *, /, %, ++, --).
Kod JavaScript programskog jezika je zanimljivo to što se decimalni brojevi ne pišu sa zarezom, već sa točkom koja razdvaja cijeli broj i decimalu. 
var x1 = 3.00;     // decimalni broj
var x2 = 34;       // cijeli broj

Object

Objekt (engl. object) je entitet koji sadrži svojstva (engl. properties) i metode (engl. methods). Svojstva su opisne karakteristike objekta, a metode su moguće radnje koje objekt može izvest.
Na primjer, ako promatramo automobil, objekt iz stvarnog svijeta, u njegove karakteristike spadaju boja, marka, broj vrata i sl. U metode spadaju njegove mogućnosti kretanja naprijed, nazad, automatsko zaključavanje vrata i sl.
Svojstva objekta se mogu smatrati varijablama koje sadrže brojeve i tekst, a metode funkcijama.
Objekti su složeni podaci, za razliku od ostalih tipova podataka koji su ovdje spomenuti. 
  var automobil = {
    marka: "Mercedes",
    brojVrata: 5,
    brojKilometara: 115.550,
    boja: "plava"
  };

Array

Array ili niz pohranjuje više vrijednosti u jednu jedinicu. Ako imate popis predmeta (na primjer popis imena automobila), pohranjivanje automobila u jednu varijablu može izgledati ovako: 
var auto1 = "Saab";
var auto2 = "Volvo";
var auto3 = "BMW";
Međutim, što ako želite proći kroz automobile i pronaći određeni? A što ako niste imali 3 automobila, već 300? Rješenje je niz! Niz može sadržavati mnoge vrijednosti pod jednim nazivom, a vrijednostima možete pristupiti pozivanjem na indeksni broj. 
var automobili = ["Saab", "Volvo", "BMW"];

NULL

Null je posebna vrsta podataka koja može imati samo jednu vrijednost: NULL. Promjenjiva vrsta podataka NULL je varijabla kojoj nije dodijeljena vrijednost. Savjet: Ako je varijabla stvorena bez vrijednosti, automatski joj se dodjeljuje vrijednost NULL. Varijable se također mogu isprazniti postavljanjem vrijednosti na NULL. Undefined tip podatka ima vrijednost indefined. Varijable koje su deklarirane, ali nisu još inicijalizirane imaju vrijednost undefined.

5. JavaScript funkcije

JavaScript funkcija je blok koda dizajniran za izvođenje određenog zadatka. JavaScript funkcija se izvršava kada se "nešto" pozove na nju. Funkcija je definirana ključnom riječi funkcije, a slijedi ime, a zatim zagrade (). Nazivi funkcija mogu sadržavati slova, znamenke, podvlake i znakove dolara (ista pravila kao i varijable). Zagrade mogu sadržavati nazive parametara razdvojenih zarezima: (parametar1, parametar2, ...) Kôd koji se izvršava, funkcijom, stavlja se u zavojite zagrade: {} Parametri funkcija navedeni su unutar zagrada () u definiciji funkcije. Argumenti funkcije su vrijednosti koje primi funkcija kad je pozvana. Unutar funkcije, argumenti (parametri) se ponašaju kao lokalne varijable. 

    function prva ()  {...}       // funkcija bez parametra

    var x = mojaPrvaFunkcija(4, 3);
    document.getElementById("demo").innerHTML = x;
    function mojaPrvaFunkcija(a, b) {
      return a * b;
      }                            // funkcija sa parametrima
Ukoliko funkcija treba vratiti (izračunatu) vrijednost, koristi se naredba return, kao u primjeru iznad.

6. Petlje

Petlje nude brz i jednostavan način da nešto ponovite. Izvršavanje koda u petlji se odvija kada se određeni kod ponavlja i izvrava sve dok traje ispunjenje određenog uvjeta.
Postoji više vrta petlji, ali sve one imaju svoju funkciju - da ponavljaju izvršavanje određenog bloka koda dokle god uvjet nije ispunjen. Čim dođe do ispunjavanja određenog uvjeta, petlja staje.
Petlje se dijele u dvije grupe:

  • one koje testiraju uvjet na početku svake iteracije - for i while petlja
  • one koje testiraju uvjet na kraju iteracije - do while petlja


for
Najčešće korištena petlja u JavaScript-u je for-petlja. Evo primjera: 
  for(inicijalizacija brojača; uvjet brojanja; inkrement brojača) {
      // neki kod koji se ponavlja
    }


  var suma = 0;
  for (var i = 1; i <= 50; i++) {
     suma = suma + i;
  }
  alert("Suma = " + suma);    // => Suma = 1275
Sastoji se od tri dijela, razdvojenih zarezima. Prvi je inicijalizator (var i = 1) koji inicijalizira petlju i izvršava se samo jednom u startu. Drugi je uvjet ispitivanja (i <= 50). Kad se uvjetni izraz ocijeni istinitim, izvršava se tijelo petlje. Kada je lažno, petlja se prekida. Treći dio je inkrement (i ++) koji se poziva nakon svake iteracije. Inkrement obično povećava ili smanjuje brojač (dekrement) petlje. Sva tri dijela, tj. inicijalizator, stanje testa i inkrement se u jednoj petlji pišu zajedno u jednom retku (koji se zove iteracijska izjava), dok se s vremenom raštrkaju i leže na različitim mjestima.

while
Ova petlja ne koristi brojač, već samo uvjet. Dokle god je uvjet točan izvršavati će se ponavaljanje. Kao i for petlja, i while petlja provjerava uvjet na početku iteracije. Oblik while petlje: 
  while (uvjet) {
    // naredbe
  }
Petlja se definira ključnom riječi while. U zagradi se navodi uvjet i u okviru vitičastih zagrada je blok koda koji pripada tijelu petlje. Nakon bloka koda se ne piše zarez. Ovaj oblik petlje se koristi kada se ne zna koliko puta se petlja treba izvesti da se ponovi kod. Primjer petlje: 
    var broj=0;
    while (broj<4) {
      console.log(broj);
      broj++;
    }
U primjeru iznad, petlja će se ponavljati dokle god je varijabla "broj" manja od 4. Nakon svake iteracije varijabla se povećava za jedan, što znači da će se petlja izvršiti 4 puta.

do-while
Ovaj oblik petlje je takav da se uvjet provjerava na kraju svake iteracije. Organizirana je slično kao i while petlja, a ono što je bitno je da će se za razliku od for i while petlje, ovaj kod izvršiti barem jednom. Njen oblik je: 
    do {
      // naredbe
    }while (uvjet);
Prvo se izvršava kod u do tijelu te se nakon toga provjerava while uvjet.
