PHP

1. Uvod u PHP programski jezik

PHP je programski skriptni jezik poslužitelja i moćan alat za izradu dinamičnih i interaktivnih web stranica.
Prije nastavka trebali biste imati osnovno razumijevanje sljedećeg:

  • HTML
  • CSS
  • JavaScript
PHP je rekurzivna skraćenica za "PHP: Hypertext Preprocessor". To je široko korišten skriptni jezik otvorenog koda. PHP skripte se izvršavaju na poslužitelju. PHP može generirati dinamični sadržaj stranice. Može stvarati, otvarati, čitati, pisati, brisati i zatvarati datoteke na serveru. PHP može prikupiti podatke iz obrasca ,slati i primati kolačiće te dodavati, brisati, mijenjati podatke u vašoj bazi podataka. PHP se može koristiti za kontrolu pristupa korisnicima te može šifrirati podatke Sa PHP-om niste ograničeni na izlazni HTML. Možete reproducirati slike, PDF datoteke, pa čak i filmove s Flashom. Također možete ispisati bilo koji tekst, poput XHTML i XML.

Zašto koristiti PHP?
  • PHP radi na različitim platformama (Windows, Mac, Linux, Raspberry Pi, itd.)
  • PHP je kompatibilan sa gotovo svim poslužiteljima koji se danas koriste (Apache, IIS itd.)
  • PHP podržava širok raspon baza podataka
  • PHP je besplatan
  • PHP je jednostavan za naučiti i učinkovito radi na strani poslužitelja
XAMPP
XAMPP je jednostavna multi-platforma s kojom instalirate Apache, MySQL, PHP, phpMyAdmin i još mnoge druge aplikacije korisne za razvoj dinamičkih web stranica izravno na vašem računalu. Instalacija je jednostavna, a sama konfiguracija minimalna. Prvo što morate napraviti je skinuti XAMPP sa službenih web stranica. Zatim pokrenite izvršnu datoteku, odaberite jezik, odredišnu mapu za instalaciju, želite prečicu na radnoj površini i u start izborniku, klin na Install te se pokreće proces instalacije. Za završetak instalacije kliknite na Finish, zatim OK.

Instalacija PHP-a
Da biste započeli koristiti PHP, možete:
Pronaći web host s PHP i MySQL podrškom
Instalirati web server na svoje računalo, a zatim instalirati PHP i MySQL.

Ako je vaš server aktivirao podršku za PHP, ne morate ništa učiniti. Samo stvorite .php datoteke, smjestite ih u vaš web direktorij i poslužitelj će ih automatski raščistiti umjesto vas. Ne morate ništa sastavljati ili instalirati dodatne alate. Budući da je PHP besplatan, većina web hostova nudi PHP podršku.

Međutim, ako vaš poslužitelj ne podržava PHP, morate:
instalirati web poslužitelj
instalirati PHP
instalirati bazu podataka, kao što je MySQL
Službena web stranica PHP-a (PHP.net) sadrži upute za instalaciju PHP-a.

2. PHP Sintaksa

Kad PHP analizira datoteku, traži otvaranje i zatvaranje oznaka, koje su Koje govore PHP-u da započne i zaustavi interpretaciju koda između njih. Parsiranje na ovaj način omogućuje PHP ugrađivanje u sve vrste različitih dokumenata, jer PHP parser zanemaruje sve izvan para oznaka otvaranja i zatvaranja. Ako datoteka sadrži samo PHP kôd, poželjno je izostaviti PHP završnu oznaku na kraju datoteke. To sprječava da se slučajno izostavi prostor ili dodaju nove linije nakon PHP oznake zatvaranja, što može prouzrokovati neželjene efekte, jer PHP će pokrenuti izlazni međuspremnik kada programer nema namjeru poslati bilo koji izlaz u toj točki skripte. 

<?php
// PHP kod piše se ovdje
?>
Zadano proširenje datoteke za PHP datoteke je ".php". PHP datoteka obično sadrži HTML oznake i neki PHP kod. Ispod imamo primjer jednostavne PHP datoteke s PHP skriptu koja koristi ugrađenu PHP funkciju "echo" za ispis teksta "Hello World!" na web stranici: 
<!DOCTYPE html?>
<html?>
<body?>

	<h1?>My first PHP page</h1?>

	<?php
	echo "Hello World!";
	?>

</body>
</html>
U PHP-u, ključne riječi (npr. if, else, while, echo, itd.), klase, funkcije i korisnički definirane funkcije razlikuju velika i mala slova. PHP interpreter ignorira sve izvan oznaka za otvaranje i zatvaranje koje u PHP datotekama imaju miješan sadržaj. To omogućava da se PHP ugradi u HTML dokumente. Kada PHP interpreter pogodi oznake ?> zatvaranja, jednostavno počinje ispuštati sve što nađe dok ne vidi drugu početnu oznaku (osim ako je usred uvjetne izjave, u tom slučaju tumač će odrediti ishod uvjeta prije donošenja odluke što će preskočiti). PHP zahtijeva da se upute na kraju svake izjave završe točka-zarez znakom. Završna oznaka bloka PHP koda automatski podrazumijeva točku-zarez; Ne trebate imati ; ako završava zadnji redak PHP bloka.

3. PHP varijable

Varijable u PHP-u predstavljene su znakom dolara, a slijedi ime varijable. Naziv varijable razlikuje velika i mala slova. Imena varijabli slijede ista pravila kao i ostale oznake u PHP-u. Važeće ime varijable započinje slovom ili podvlakom, nakon čega slijedi bilo koji broj slova, brojeva ili podvlaka. 


<?php
	$ var = 'jabuka';
	$ var = 'banana';
	echo "$ var, $ var";

	$2voce = 'dinja'
	$_2voce = 'jagoda';
	$voće = 'marelica';
?>
Prema zadanim postavkama, varijable su uvijek dodijeljene po vrijednosti. To jest, kad dodijelite izraz varijabli, cijela se vrijednost izvornog izraza kopira u odredišnu varijablu. To, primjerice, znači da nakon dodjeljivanja vrijednosti jedne varijable drugoj, promjena jedne od tih varijabli neće imati utjecaja na drugu. PHP nudi i drugi način dodjeljivanja vrijednosti varijablama: dodijeliti referencom. To znači da nova varijabla jednostavno upućuje na izvornu varijablu. Promjene nove varijable utječu na izvornike i obrnuto.

Pravila za PHP varijable:
  • Varijabla započinje znakom $, a slijedi naziv varijable
  • Naziv varijable mora započeti slovom ili znakom podvlake
  • Ime varijable ne može započeti brojem
  • Naziv varijable može sadržavati samo alfa-numeričke znakove i podvlake (A-z, 0-9 i _)
  • Imena varijabli razlikuju velika i mala slova ($ age i $ AGE su dvije različite varijable)
U PHP-u, varijable se mogu deklarirati bilo gdje u skripti. Opseg varijable je dio skripte u kojem se varijabla može uputiti / koristiti.

PHP ima tri različita opsega varijabli:
  • lokalno
  • globalno
  • statički


    • <?php
		$a = 1; /* globalno */
	function test()
		{
			echo $a; /* lokalno */
		}
	test();
?>

4. PHP tipovi podataka

Varijable mogu pohraniti podatke različitih vrsta, a različite vrste podataka mogu činiti različite stvari.
PHP podržava sljedeće vrste podataka:

  • String
  • Integer
  • Float
  • Boolean
  • Array
  • Object
  • NULL
  • Resource


String

String je niz znakova, poput "Hello world!". String može biti bilo koji tekst unutar citata. Možete koristiti jednostruke ili dvostruke navodnike. 
<?php
	$x = "Hello world!";
	$y = 'Hello world!';

	echo $x;
	echo "<br>";
	echo $y;
?>


Integer

Integer je broj između -2,147,483,648 i 2,147,483,647.
Pravila za integer:
  • Integer mora imati najmanje jednu znamenku
  • Integer ne smije imati decimalnu točku
  • Integer može biti pozitivan ili negativan
  • Integer se može odrediti u tri formata: decimalni (10-baziran), šesterokutni (16-baziran - s prefiksom 0x) ili oktalni (8-baziran - s prefiksom 0)
PHP var_dump () funkcija vraća vrstu i vrijednost podataka 
<?php
	$x = 5985;
	var_dump($x);
?>


Float

Float je broj s decimalnom točkom. U sljedećem primjeru $ x je float. PHP var_dump () funkcija vraća vrstu i vrijednost podataka 
<?php
	$x = 10.365;
	var_dump($x);
?>


Boolean

Boolean predstavlja dva moguća stanja: TRUE ili FALSE. 
<?php
	$ x = true;
	$ y = false;
?>


Array

Array ili niz pohranjuje više vrijednosti u jednu jedinicu. U sljedećem primjeru $ gradovi je niz. PHP var_dump () funkcija vraća vrstu i vrijednost podataka: 
<?php
	$gradovi = array("Zagreb","Rijeka","Osijek");
	var_dump($gradovi);
?>


Object

Object je vrsta podataka koja pohranjuje podatke i informacije o načinu obrade tih podataka. U PHP-u se objekt mora izričito deklarirati. Prvo moramo proglasiti klasu objekta. Za to koristimo ključnu riječ class. Klasa je struktura koja može sadržavati svojstva i metode. 
<?php class student
	{
	function izracun_ocjene()
	{         
	echo "Prikazi ocjene";
	}
	}
	$student1 = new student;
	$student1->izracun_ocjene); 
?>


NULL

Null je posebna vrsta podataka koja može imati samo jednu vrijednost: NULL. Promjenjiva vrsta podataka NULL je varijabla kojoj nije dodijeljena vrijednost. Savjet: Ako je varijabla stvorena bez vrijednosti, automatski joj se dodjeljuje vrijednost NULL. Varijable se također mogu isprazniti postavljanjem vrijednosti na NULL. 
<?php
	$x = "Hello world!";
	$x = null;
	var_dump($x);
?>


Resource

Ovo nije stvarna vrsta podataka. To je spremanje referenci na funkcije i resurse izvan PHP-a. Čest primjer korištenja vrste podataka resursa je poziv iz baze podataka.

5. PHP petlje

Petlje u PHP-u koriste se za izvršavanje istog bloka koda određeni broj puta.
PHP podržava sljedeće četiri vrste petlje.

  • for - prolazi kroz blok koda određeni broj puta.
  • while - prolazi kroz blok koda ako je i pod uvjetom da je navedeni uvjet istinit.
  • do ... while - jednom prolazi kroz blok koda, a zatim ponavlja petlju sve dok je poseban uvjet istinit.
  • foreach - prolazi kroz blok koda za svaki element u nizu.
FOR 
<?php
	$a = 0;
         $b = 0;
         for( $i = 0; $i<5; $i++ ) {
            $a += 10;
            $b += 5;
         }
         echo ("Na kraju petlje a = $a i b = $b" );
?>
WHILE 
<?php
	 $i = 0;
         $br = 50;
         while( $i < 10) {
            $br--;
            $i++;
         }
         echo ("Petlja staje na i = $i i br = $br" );
?>
DO WHILE 
<?php
	$i = 0;
         $br = 0;
         do {
            $i++;
         }
         while( $i < 10 );
         echo ("Petlja staje na i = $i" );
?>
FOREACH 
<?php
	$niz = niz( 1, 2, 3, 4, 5);
         foreach( $niz as $vrijednost ) {
            echo "Vrijednost je $vrijednost 
";
         }
?>
IF
Izjava if izvršava blok koda ako je jedan uvjet istinit. 
<?php
	if ($a > $b) {
  echo "a je veće od b";
  $b = $a;
}
?>
IF...ELSE
Izjava if ... else izvršava neki blok koda ako je jedan uvjet istinit, a drugi blok koda ako je taj uvjet neistinit. 
<?php
if ($a > $b) {
  echo "a je veće od b";
} else {
  echo "a nije veće od b";
}
?>
IF...ELSEIF...ELSE
Izjava if ... elseif ... else izvršava različite blokove koda za više od dva uvjeta. 
<?php
if ($a > $b) {
    echo "a je veće od b";
} elseif ($a == $b) {
    echo "a je jednako b";
} else {
    echo "a je manje od b";
}
?>

6. PHP niz

Nizovi se koriste za pohranjivanje više vrijednosti u jednu varijablu. 

<?php
	$gradovi = array("Zagreb", "Osijek", "Rijeka");
	echo "Bila sam u " . $gradovi[0] . ", " . $gradovi[1] . " i " . $gradovi[2] . ".";
?>
U PHP-u, funkcija array () koristi se za stvaranje niza: 
array();
U PHP-u postoje tri vrste nizova:
  • Indeksirani nizovi - nizovi s numeričkim indeksom
  • Asocijativni nizovi - Nizovi s imenovanim ključevima
  • Višedimenzionalni nizovi - Nizovi koji sadrže jedan ili više nizova


Indeksirani niz

Postoje dva načina za stvaranje indeksiranih nizova: Indeks se može dodijeliti automatski (indeks uvijek započinje s 0), ovako: 
<?php
	$gradovi = array("Zagreb", "Osijek", "Rijeka");
?>
ili se indeks može dodijeliti ručno: 
<?php
	$gradovi[0] = "Zagreb";
	$gradovi[1] = "Osijek";
	$gradovi[2] = "Rijeka";
?>
Funkcija count () koristi se za vraćanje duljine (broja elemenata) niza. 
<?php
	$gradovi = array("Zagreb", "Osijek", "Rijeka");
	echo count($gradovi);
?>
Da biste pregledali i ispisali sve vrijednosti indeksiranog niza, mogli biste upotrijebiti petlju, poput ove: 
<?php

Asocijativi niz

Asocijativni nizovi su nizovi koji koriste imenovane ključeve koje im dodijelite. Postoje dva načina za stvaranje asocijativnog niza: 
<?php
	$dob = array("Petar"=>"35", "Maja"=>"37", "Katarina"=>"43");
	?>
ili: 
<?php
	$dob['Petar'] = "35";
	$dob['Maja'] = "37";
	$dob['Katarina'] = "43";
?>
Da biste pregledali i ispisali sve vrijednosti asocijativnog niza, mogli biste koristiti petlju foreach-a, poput ove: 
<?php
	$dob = array("Petar"=>"35", "Maja"=>"37", "Katarina"=>"43");

	foreach($dob as $x => $x_value) {
		echo "Key=" . $x . ", Value=" . $x_value;
		echo "< br >";
	}
?>


Višedimenzionalni niz

Višedimenzionalni niz je niz koji sadrži jedan ili više nizova. PHP razumije višedimenzionalne nizove duboke dvije, tri, četiri, pet ili više razina. Međutim, nizovi dublji od tri razine teško su razumljivi većini ljudi.
