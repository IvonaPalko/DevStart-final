CSS

1. Što je CSS?

CSS, skraćenica od Cascading Style Sheets, predstavlja tekstualni dokument koji sadrži instrukcije za formatiranje namijenjene pretraživačima o određenim elementima, odjeljcima ili stranama web lokacija. Instrukcije se primjenjuju na elemente koji su obuhvaćeni u skupu instrukcija, pri čemu je nebitno gdje se one pojavljuju na web stranici.

Alati za uređivanje CSS-a

Osnovna "oprema" za rad sa CSS-om, kao i HTML-om, je alat za uređivanje koda i teksta; uz naravno, računalo i pristup Internetu.

2. Definiranje CSS naredbi

CSS kod se sastoji od nekoliko izdvojenih komponenti koje formiraju njegova pravila ili instrukcije. Selektor (engl. selector) deklarira koje će elemente formatiranje primjeniti, svojstvo (engl. property) koji dio elementa će biti formatiran, a vrijednost (engl. value) govori koje je formatiranje primijenjeno na element.
Objašnjenje:

Oznaka h1 predstavlja element iz HTML datoteke, odnosno sve naslove veličine 1 HTML datoteke na koju se ovaj CSS odnosi. Prema toj naredbi, svi naslovi koji su označeni elementom h1 u pojedinom HTML dokumentu će biti obojani u crveno.

3. Selektori

Selektorima se nazivaju načini dohvaćanja HTML elementa kojeg se želi stilizirati. Postoji više načina, i svaki je najbolji za pojedinu svrhu. Primjer prikazan gore (sa oznakom h1) je primjer selektora prema elemenetu jer uzima pojedinu oznaku i na njoj vrši izmjene.

    3. - dodavanje klase elementima i CSS označavanje
    4. - simbol * - stilizira cijelu stranicu (npr. pozadina)
    5. - okupljanje više elemenata za koje se vrši ista stilizacija

4. CSS svojstva

Kako ne bismo prolazili kroz sva CSS svojstva jer ih je veoma mnogo, a jednom kada počnete raditi s njima postati će vam vrlo znane, ovdje ćemo priložiti tek izvore sa popisom svojstava i potencijalnih vrijednosti.

5. Boje

CSS boje se definiraju pomoću modela za opis boja RGB (engl. Red, Green, Blue), heksadecimalnog sistema ili po imenu (na engleskom).
U ovom primjeru ćemo raditi s bojom fuksija. Fuksija je kao boja kombinacija crvene i plave boje, u ravnomjernom omjeru. Također, kreiranje boja u ovim oblicima je apsolutno logično:

RGB
Postoje dva načina rada sa RGB modelom:

    Prvi način je upisivanje postotka crvene, zelene i plave
    color: rgb (100%, 0%, 100%);
    a drugi način je direktno definiranje RGB vrijednosti boja
    color: rgb(255,0,255);
Heksadecimalni sistem
    color: #FF00FF;
Korištenje imena boje
    color: cornflowerblue;

6. Fontovi i tekst

Fontovi koji se koriste na web stranici trebaju se nalaziti u većini operativnih sustava. Ukoliko korisnikov operativni sustav nema instaliran font koji web stranica definira, preglednik će prikazati korisnikov skup standardnih fontova, što predstavlja problem zbog potrošenog vremena u kreiranje ili odabir fonta kojeg korisnik na kraju neće vidjeti. 

  font-family  definira obitelj fontova koja će se koristiti (može biti određeno ime ili opći naziv
               obitelji)
  font-style   definira stil fonta koji može biti normalan, kurziv ili ukošen
  font-weight  definira debljinu fonta
  font-size    definira veličinu fonta
  font         postavlja obitelj, stil, debljinu i veličinu fonta jednom deklaracijom

7. Poravnanja

Svaki element sam po sebi ima određeno poravnanje, dok se ono može (i treba) dodatno uređivati prema pravilima koja postavimo za web stranicu koju kreiramo. Termin model okvira (engl. box) u CSS-u predstavlja model u kojem se postavljaju elementi na web stranici kao polja sastavljena od rubova, margina i razmaka od rubova elemenata. Određeni elementi se mogu precizno postaviti na željenu stranu i tako napraviti logičan, prirodan slijed sadržaja.

Interna udaljenost elemenata, tj. razmak između sadržaja elementa i njegovih rubova postavlja se pomoću CCS svojstva padding. Kako biste postavili udaljenost od rubova, potrebno je definirati gornji, donji, lijevi i desni razmak u vidu izdvojenih deklaracija: 

      p {
        padding-top: 2px;       definira udaljenost od gornjeg ruba elementa
        padding-bottom: 3px;    definira udaljenost od donjeg ruba elementa
        padding-left: 3px;      definira udaljenost od lijevog ruba elementa
        padding-right: 4px;     definira udaljenost od desnog ruba elementa
      }

      ili kroz pojedinačnu deklaraciju:

      p {padding: 2px 3px 3px 4px;}

Nadalje, kako bismo koristili razmak između elemenata koristi se margina. Ona služi za definiranje prostora između rubova elemenata koju niti jedan drugi element ne smije dodirivati ili prelaziti preko njega. Na primjer, ukoliko dva elementa imaju marginu od 5px, ukupan razmak između ta dva elementa biti će 10px.
Kao i kod paddinga, elementi imaju desnu, lijevu, gornju i donju marginu te se mogu definirati kao odvojeno ili kao jedna deklaracija. Širina margine može biti dužina (u pikselima), postotak ili automatski postavljena. 

      p {
        margin-top: 2px;       definira gornju marginu elementa
        margin-bottom: 3px;    definira donju marginu elementa
        margin-left: 3px;      definira lijevu marginu elementa
        margin-right: 4px;     definira desnu marginu elementa
      }

      ili kroz pojedinačnu deklaraciju:

      p {margin: 2px 3px 3px 4px;}

Posljednja stavka u box modelu je border, a on se nalazi između prostora koji rješavamo paddingom i prostora margina. Border se, kao i prijašnja dva svojstva, označava sa sve četiri strane. 

      p {
        border-top: 2px;       definira gornji obrub elementa
        border-bottom: 3px;    definira donji obrub elementa
        border-left: 3px;      definira lijevi obrub elementa
        border-right: 4px;     definira desni obrub elementa
      }

      ili kroz pojedinačnu deklaraciju:

      p {border: 2px 3px 3px 4px;}

8. Pozicioniranje elemenata

CSS osigurava instrukcije za raspored elemenata u apsolutne, relativne ili fiksne položaje u odnosu na ostale elemente. U maloprijašnje spomenutom modelu bloka (engl. box) govorilo se o elementima posmatranim kao blokovima na web stranici. Ti isti blokovi su ono što se pozicionira na web stranici.

Svojstvo float pomjera element što je dalje moguće udesno ili ulijevo unutar matičnog elementa. Ostali elementi se raspoređuju oko tog plutajućeg elementa, nalijevo ili nadesno od njih. Definicija float: left postavlja elemente desno od elementa kojemu je dodana ta vrijednost. 

 float      definira element koji se treba postaviti sa jedne ili druge strane svih ostalih elemenata
 - Left       definira element koji pluta sa lijeve strane ostalih elemenata
 - Right      definira element koji pluta sa desne strane ostalih elemenata
 - None       definira element koji ne pluta
 - Inherit    definira element koji je naslijedio vrijednost plutanja od njemu nadređenog elementa

 clear      definira jednu ili dvije strane u koje se ne mogu postaviti plutajući elementi
 - Both       definira plutajuće elemente koji ne mogu plutati nalijevo ili nadesno od nekog elementa
 - left       definira da niti jedan element ne može plutati lijevo od nekog elementa
 - right      definira da niti jedan element ne može plutati desno od nekog elementa
 - none       dozvoljava elementima da plutaju lijevo, desno ili sa obje strane nekog elementa
 - inherit    definira element bloka koji nasljeđuje samo vrijednost naređenog u kome je sadržan
